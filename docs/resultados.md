# 📊 Resultados Esperados — LeanCity

## 📌 Visão Geral

Este documento apresenta os principais resultados esperados com o desenvolvimento do projeto LeanCity — Sistema de Eficiência Energética com IoT.

O projeto tem como objetivo demonstrar como sensores, automação e sistemas embarcados podem ser utilizados para reduzir desperdícios de energia, melhorar o controle de ambientes e apoiar soluções voltadas a cidades inteligentes.

---

## 🎯 Objetivo dos Resultados

O objetivo principal é validar uma solução capaz de monitorar condições ambientais e energéticas, interpretar os dados coletados e acionar dispositivos automaticamente para melhorar a eficiência energética.

---

## ✅ Resultados Funcionais Esperados

Ao final do projeto, espera-se que o sistema consiga:

* Detectar presença no ambiente
* Ler níveis de luminosidade
* Simular ou monitorar temperatura
* Simular consumo energético
* Processar dados no microcontrolador
* Acionar iluminação quando necessário
* Desligar cargas quando não houver necessidade
* Emitir alerta em caso de consumo elevado
* Exibir informações no monitor serial
* Demonstrar aplicação prática de IoT e automação

---

## ⚡ Ganhos Relacionados à Eficiência Energética

A solução proposta pode contribuir para:

* Redução do desperdício de energia
* Uso inteligente da iluminação
* Menor dependência de acionamento manual
* Melhor controle sobre dispositivos elétricos
* Identificação de consumo elevado
* Otimização do uso de recursos
* Apoio à sustentabilidade tecnológica

---

## 🏙️ Ganhos para Cidades Inteligentes

O projeto representa uma aplicação simplificada de smart city, demonstrando como sensores e automação podem melhorar a gestão de ambientes.

Possíveis ganhos:

* Ambientes mais eficientes
* Monitoramento automatizado
* Redução de desperdícios
* Uso de dados para tomada de decisão
* Maior sustentabilidade
* Possibilidade de expansão para prédios inteligentes

---

## 📊 Comparação entre Processo Manual e Sistema Automatizado

| Critério                      | Processo Manual       | Sistema LeanCity         |
| ----------------------------- | --------------------- | ------------------------ |
| Controle de iluminação        | Manual                | Automatizado             |
| Detecção de presença          | Não automatizada      | Feita por sensor         |
| Monitoramento de luminosidade | Inexistente ou manual | Automatizado             |
| Controle de consumo           | Limitado              | Monitorado pelo sistema  |
| Alerta de consumo elevado     | Não disponível        | Acionado automaticamente |
| Eficiência energética         | Menor                 | Maior                    |
| Tomada de decisão             | Humana                | Baseada em dados         |
| Sustentabilidade              | Limitada              | Apoiada por automação    |

---

## 🧪 Resultados em Simulação

Durante a simulação, espera-se observar:

* Leitura dos sensores no monitor serial
* Alteração dos valores de luminosidade, temperatura e consumo
* Acionamento da iluminação quando houver presença e baixa luminosidade
* Desligamento da iluminação quando não houver necessidade
* Acionamento do buzzer quando o consumo estiver elevado
* Registro das mensagens de status do sistema

## ✅ Conclusão

O projeto LeanCity demonstra como uma solução simples, baseada em sensores e automação, pode contribuir para a eficiência energética e a sustentabilidade.

Mesmo em uma versão acadêmica e demonstrativa, o sistema mostra potencial para aplicação em ambientes reais, como salas, laboratórios, escritórios, prédios inteligentes e projetos de cidades inteligentes.
