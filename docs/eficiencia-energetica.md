# ⚡ Eficiência Energética — LeanCity

## 📌 Visão Geral

Este documento apresenta o conceito de eficiência energética aplicado ao projeto LeanCity — Sistema de Eficiência Energética com IoT.

A proposta do projeto é demonstrar como sensores, automação e sistemas embarcados podem contribuir para reduzir desperdícios de energia em ambientes urbanos, acadêmicos, comerciais e residenciais.

---

## 🎯 Objetivo

O objetivo da eficiência energética no projeto LeanCity é utilizar a energia de forma mais inteligente, acionando dispositivos apenas quando necessário e monitorando condições que podem indicar desperdício.

A solução busca mostrar como a tecnologia pode apoiar ambientes mais sustentáveis, econômicos e automatizados.

---

## ⚡ O que é Eficiência Energética?

Eficiência energética significa utilizar menos energia para realizar a mesma função, reduzindo desperdícios sem comprometer o funcionamento do ambiente.

No contexto do LeanCity, isso significa:

* Ligar iluminação somente quando necessário
* Desligar cargas quando não houver presença
* Monitorar consumo elevado
* Utilizar sensores para tomada de decisão
* Automatizar ações que normalmente seriam manuais

---

## 🧠 Como o LeanCity contribui para a economia de energia

O sistema utiliza sensores para identificar as condições do ambiente e tomar decisões automáticas.

Exemplos:

* Se não houver presença no ambiente, a iluminação pode permanecer desligada.
* Se houver presença, mas a luminosidade natural for suficiente, a iluminação artificial não precisa ser acionada.
* Se o consumo energético estiver elevado, o sistema pode emitir um alerta.
* Se houver necessidade real de uso, o sistema aciona a carga automaticamente.

---

## 🔄 Lógica de Controle Energético

A lógica de controle do projeto pode ser representada da seguinte forma:

Presença detectada + baixa luminosidade = iluminação ligada

Presença detectada + boa luminosidade = iluminação desligada

Sem presença = iluminação desligada

Consumo elevado = alerta acionado

Temperatura elevada = aviso no sistema
