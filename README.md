# 🏙️ LeanCity — Sistema de Eficiência Energética com IoT

## 📌 Sobre o Projeto

O LeanCity é um projeto voltado ao desenvolvimento de um protótipo de cidade inteligente com foco em eficiência energética, automação e sustentabilidade tecnológica.

A solução utiliza conceitos de Internet das Coisas para monitorar variáveis como presença, temperatura, luminosidade e consumo energético, permitindo o acionamento inteligente de dispositivos e a otimização do uso de recursos.

O projeto foi desenvolvido no contexto acadêmico da UniFECAF, aplicando conhecimentos de sistemas embarcados, sensores, automação, análise de dados, sustentabilidade e cidades inteligentes.

## 🧠 Problema Identificado

Em ambientes urbanos, prédios, salas, laboratórios e espaços públicos, é comum que luzes, equipamentos e sistemas elétricos permaneçam ligados mesmo sem necessidade.

Esse desperdício gera aumento no consumo energético, custos operacionais maiores e impacto ambiental negativo.

A ausência de monitoramento inteligente dificulta a identificação de pontos de desperdício e reduz a eficiência no uso dos recursos.

---

## 💡 Solução Proposta

A solução proposta utiliza sensores e automação para monitorar condições do ambiente e controlar dispositivos de forma inteligente.

O sistema pode identificar presença, luminosidade e temperatura, permitindo acionar ou desligar cargas automaticamente conforme a necessidade.

A proposta simula uma aplicação de cidade inteligente, onde sensores coletam dados, o sistema interpreta as informações e os dispositivos são controlados para aumentar a eficiência energética.

## 🔄 Funcionamento do Sistema

1. Os sensores coletam informações do ambiente.
2. O microcontrolador processa os dados recebidos.
3. O sistema verifica condições de presença, luminosidade e consumo.
4. Caso não haja presença ou exista luminosidade suficiente, a carga pode ser desligada.
5. Caso o ambiente necessite de iluminação ou climatização, o dispositivo pode ser acionado.
6. Os dados podem ser registrados para análise posterior.
7. O sistema contribui para redução do consumo energético e melhor uso dos recursos.

## 🚨 Automação Inteligente

O sistema pode tomar decisões automáticas com base nas leituras dos sensores.

Exemplos:

* Desligar iluminação quando não houver presença.
* Acionar iluminação quando o ambiente estiver escuro.
* Monitorar temperatura para apoiar controle de climatização.
* Registrar consumo energético para análise.
* Emitir alerta em situação de consumo elevado.
